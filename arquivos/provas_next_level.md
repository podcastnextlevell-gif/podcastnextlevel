## Prova - Temporada 0: Episódios Introdutórios

### Questão 1
Qual é a principal mudança de mentalidade exigida do Analista Next Level, conforme discutido no EP00A?
A) Focar apenas em relatórios de TMA e FCR.

**Justificativa:** O Analista Next Level deve evoluir de um mero reportador de métricas para um profissional que usa dados (SQL, Python, IA) para projetar e implementar soluções estratégicas, tornando-se um "arquiteto da qualidade".

**Justificativa:** A fase de Fundamentos visa estabelecer a base analítica, focando no domínio de SQL Básico para extração de dados e Estatística Descritiva para a compreensão inicial das métricas.

### Questão 5
